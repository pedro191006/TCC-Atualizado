Conclusão

Este trabalho teve como objetivo o desenvolvimento de um sistema de gerenciamento de banco de dados para uma aplicação de gestão de inventário, utilizando a abordagem de modelagem relacional para garantir a integridade e eficiência no armazenamento e recuperação dos dados. Durante o desenvolvimento, foi possível aplicar os conceitos estudados ao longo do curso, como normalização, otimização de consultas e uso de índices, resultando em um banco de dados bem estruturado e funcional.

O estudo demonstrou que a escolha da tecnologia adequada é essencial para garantir a escalabilidade e a performance do sistema, sendo que o MySQL se mostrou uma excelente opção para o contexto deste trabalho devido à sua robustez, compatibilidade com sistemas de alto desempenho e facilidade de uso. A implementação de procedimentos armazenados e triggers contribuiu para a automação de processos internos, reduzindo a necessidade de interação manual e aumentando a segurança e a integridade dos dados. Além disso, ao aplicar as boas práticas de modelagem, foi possível evitar redundância de dados e garantir a consistência referencial, o que representa um avanço significativo no controle do inventário, atendendo de forma eficiente às necessidades da organização simulada no estudo de caso.

Por fim, é importante destacar que, embora os resultados tenham sido satisfatórios, algumas limitações foram encontradas durante o desenvolvimento, como a necessidade de maior análise sobre questões relacionadas à segurança e performance em sistemas com maior volume de dados. Para trabalhos futuros, sugere-se a implementação de soluções de backup automatizado, análise de dados em tempo real e a exploração de bancos de dados NoSQL, que podem ser mais adequados para determinados tipos de aplicações que exigem alta escalabilidade e flexibilidade.

Este estudo, portanto, cumpre seu objetivo de contribuir para o aprimoramento da gestão de dados em sistemas empresariais, ao mesmo tempo em que serve como ponto de partida para pesquisas mais aprofundadas sobre a evolução das tecnologias de banco de dados e seus impactos no desenvolvimento de soluções mais eficazes.
