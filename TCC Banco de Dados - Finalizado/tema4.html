Linguagens de Consulta

4.1 SQL (Structured Query Language/Linguagem Estruturada de Consultas)

SQL é a linguagem padrão dos bancos de dados relacionais, Ela tem o objetivo de facilitar a forma de modificar os dados de uma empresa

4.2 Principais comandos e operadores

O SQL tem 4 subgrupos de linguagem, o DDL (Data Definition Language) que comandam a modificação do banco de dados DCL (Data Control Language) que controla o acesso dos dados em um banco de dados DML (Data Manipulation Language) esses comandos são respoonsaveis pela alteração das informações nas tabelas DTL (Data Transaction Language) Elas gerenciam e fazem fazer transações ocorridas dentro de um banco de dados Seus principais comandos são: CREATE = Cria uma nova tabela ALTER = Altera uma tabela já existente UPGRADE = Atualiza os dados DELETE = Remove os dados selecionados SELECT = Permite ao usuário especificar exatamente quais dados deseja recuperar. USE = Seleciona o Banco de dados INSERT = Insere novos dados ao banco DROP = Remove uma tabela
